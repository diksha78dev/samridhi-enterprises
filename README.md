# 🚗 Samridhi Enterprises - Vehicle Spare Parts E-Commerce

**Samridhi Enterprises** is a full-stack e-commerce web application tailored for managing and selling **vehicle spare parts**. This project is currently under development as part of a freelance contract.

---

## 📌 Project Status

🛠️ **Under Development**  
We are actively building features, optimizing performance, and finalizing UI/UX.

---

## 🔧 Tech Stack

### 🌐 Frontend (`SE/client`)
- React.js + Vite
- Redux Toolkit
- Tailwind CSS
- Framer Motion
- Axios
- React Toastify

### 🖥️ Backend (`SE/server`)
- Node.js
- Express.js
- MongoDB + Mongoose
- Cloudinary (for image storage)
- JWT (Authentication)
- Brevo (Email service)

---

## 🔐 Environment Variables

Environment variables are kept private using `.env` files.
